# EMSCRIPTEN SDL GL GUI test

This is an attempt at getting a WebGL GUI using emscripten with c(++) up and running

## Requirements:
- make
- git
- emscripten

## How to compile:
Just `make` it.

## How to run:
Open `out/index.html` in a modern browser
